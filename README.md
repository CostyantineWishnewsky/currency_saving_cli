# Currency Saving CLi

This is cli application and client of Currency Saving Api.


## Description
This app can connect to the api and gather currencies exchange rates and display them to the terminal.This is example of using MVC pattern.
## Getting Started

### Dependencies
* Python 3.9.2 or higher


### Installing

#### For Linux
#### Clonning
```
git clone https://github.com/CostyantineWishnewsky/currency_saving_cli.git
```



### Executing program
```
python3 __main__.py
```
